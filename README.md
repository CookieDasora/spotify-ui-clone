# Spotify UI Clone

A Spotify UI Clone made with SvelteKit and TailwindCSS.

![App Screenshot](https://i.imgur.com/XUIOg2T.png)


## Running it!

Clone the project

```bash
  git clone https://github.com/CookieDasora/spotify-ui-clone.git
```

Go to the directory

```bash
  cd spotify-ui-clone
```

Install the dependencies

```bash
  npm install
```

And start it!

```bash
  npm run dev
```
